# 🏭 TIA Portal SCL Code Generator

Генератор кода для SIMATIC TIA Portal (SCL) из Excel файлов системы ввода-вывода.

## 📋 Описание

Программа автоматически парсит Excel файлы с таблицами ШС (Шкаф Автоматики) и ШСАУ, 
анализирует структуру оборудования и генерирует готовый код на языке SCL для TIA Portal.

## 🎯 Возможности

- ✅ Парсинг Excel файлов с листами ШС и ШСАУ
- ✅ Поддержка формул Excel (=$A$7)
- ✅ Автоматическое определение типов устройств по сигналам
- ✅ Генерация кода SCL с REGION блоками
- ✅ Создание секций для AI, AO, DI, DO модулей
- ✅ Инициализация флагов по умолчанию (WHILE циклы)
- ✅ Каскадные связи для доливов
- ✅ **Графический интерфейс (GUI)**
- ✅ **Веб-интерфейс** с drag & drop
- ✅ Статистика генерации

## 🔧 Настройка

### Формат входного Excel файла

Программа ожидает Excel файл со следующими листами:
- **ШС** - Шкаф Автоматики (основной)
- **ШСАУ** - Шкаф Автоматики и Управления (дополнительный)

### Структура таблицы

| Оборудование | Устройство | Сигнал | Адрес | Модуль | ... |
|-------------|-----------|--------|-------|--------|-----|
| Cart        | Cart1     | Load   | %I0.0 | DI     | ... |

## 📊 Примеры использования

### Генерация кода для всех листов

```bash
python ExcelToSCL/excel_to_scl.py "МойПроект.xlsx" -o result.txt
```

### Генерация только для конкретного листа

## ❓ FAQ

### Q: Программа не запускается с ошибкой "tkinter not found"
**A:** Установите tkinter:
- Ubuntu/Debian: `sudo apt-get install python3-tk`
- Fedora: `sudo dnf install python3-tkinter`
- macOS: `brew install python-tk`

Или используйте веб-интерфейс: `./run_web_gui.sh`

### Q: Как добавить поддержку нового типа оборудования?
**A:** Откройте `excel_to_scl.py` и добавьте новый тип в метод `_determine_device_type()`.

### Q: Можно ли изменить формат выходного кода?
**A:** Да, отредактируйте шаблоны генерации в методах класса `ExcelToSCLConverter`.
